# Vendored fonts

## ChicagoFLF.woff2

Source: https://github.com/mihaip/infinite-mac/blob/main/src/Fonts/ChicagoFLF.woff2

Vendored to avoid a runtime dependency on an external CDN font URL and to prevent
browser console 404s when loading the docs browser.

The source repository is licensed under the Apache License 2.0:
https://github.com/mihaip/infinite-mac/blob/main/LICENSE

## EB Garamond (eb-garamond-latin-*.woff2)

Source: @fontsource/eb-garamond npm package
https://github.com/fontsource/font-files/tree/main/fonts/eb-garamond

EB Garamond is based on the Garamond typeface design, digitized by Georg Duffner.
Licensed under the SIL Open Font License 1.1:
https://github.com/google/fonts/blob/main/ofl/ebgaramond/OFL.txt

Subset files included: latin and latin-ext, weights 400 (normal + italic), 500, 600, 700.
