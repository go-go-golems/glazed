You can ask the help system to list all toplevel topics (not just the default ones) in 
a concise list.

---

```
❯ glaze help --list                    

   glaze - glaze is a tool to format structured data                                                   
                                                                                                       
  --------                                                                                             
                                                                                                       
  For more help, run:  glaze help glaze
```
