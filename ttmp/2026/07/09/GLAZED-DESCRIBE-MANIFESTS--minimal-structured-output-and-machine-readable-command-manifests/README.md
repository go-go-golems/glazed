## Getting Started

Use docmgr commands to manage this workspace:

- Add documents: `docmgr doc add --ticket GLAZED-DESCRIBE-MANIFESTS --doc-type design-doc --title "My Design"`
- Import sources: `docmgr import file --ticket GLAZED-DESCRIBE-MANIFESTS --file /path/to/doc.md`
- Update metadata: `docmgr meta update --ticket GLAZED-DESCRIBE-MANIFESTS --field Status --value review`
